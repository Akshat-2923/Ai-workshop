# Incredible India Trails

A responsive travel website focused on Indian destinations, built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` - Home page
- `destinations.html` - Destination listing with search and filters
- `destination.html` - Dynamic destination details page using query params
- `about.html` - Website story and mission
- `contact.html` - Contact form and social links

## Features

- Responsive modern UI
- Light and dark mode toggle
- Smooth reveal animations
- Destination search
- Filters by state, type, and budget
- Save to favorites with `localStorage`
- Shared data-driven destination content

## Run locally

Open `index.html` directly in a browser, or serve the folder with a simple local server if you prefer.

Example with Python:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
